# f5devcentral/.github

This repository provides org-wide defaults for the [f5devcentral](https://github.com/f5devcentral) GitHub organization.

## What's in here

**Community health files** (inherited by any repo in the org that doesn't have its own copy):

- [`CODE_OF_CONDUCT.md`](CODE_OF_CONDUCT.md)
- [`CONTRIBUTING.md`](CONTRIBUTING.md)
- [`SECURITY.md`](SECURITY.md)
- [`SUPPORT.md`](SUPPORT.md)

**Issue and PR templates** (inherited by any repo without its own):

- [`.github/ISSUE_TEMPLATE/`](.github/ISSUE_TEMPLATE/)
- [`.github/pull_request_template.md`](.github/pull_request_template.md)

**Org landing page:**

- [`profile/README.md`](profile/README.md) renders at <https://github.com/f5devcentral>

## How inheritance works

GitHub automatically applies the files above as fallbacks for any repo in the f5devcentral org that doesn't have its own copy. Repos with their own files override these defaults. Workflows do **not** inherit and must exist per-repo.

For more on GitHub's community health file behavior, see [GitHub's documentation](https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file).

## Contributing

Changes to this repository affect every repo in the org. Treat with appropriate care:

- Open a pull request for any changes; do not push directly to `main`
- Request review from at least one other maintainer
- Significant policy changes (Code of Conduct, Security reporting, AI tool policy) require broader review beyond this repo's maintainers
